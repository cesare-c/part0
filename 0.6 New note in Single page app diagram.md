# Exercises 0.6: New note in Single page app diagram

Create a diagram depicting the situation where the user creates a new note using the single-page version of the app.


```mermaid
sequenceDiagram
    participant browser
    participant server

    Note over browser: User types a new note<br>and clicks "Save"

    Note right of browser: SPA JavaScript intercepts the form submit<br>and prevents full page reload

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa<br>with JSON: { content, date }
    activate server
    Note over server: Server stores the new note<br>and updates data.json
    server-->>browser: 201 Created (JSON confirmation)
    deactivate server

    Note right of browser: SPA updates UI immediately<br>by adding the new note to the list<br>without reloading the page


```
